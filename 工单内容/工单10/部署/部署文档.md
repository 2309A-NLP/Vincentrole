# 工单10 — 金融问答系统部署

## 一、部署概述

| 项目 | 内容 |
|------|------|
| 系统 | 金融问答 RAG 系统（基于招股说明书） |
| 基础镜像 | python:3.11-slim |
| 模型 | bge-small-zh-v1.5（本地嵌入） |
| LLM | 通义千问 qwen-turbo（API） |
| 镜像大小 | 3.39GB（含模型+索引+数据） |
| 部署方式 | Docker 容器 |
| 端口映射 | 宿主机:8520 → 容器:8501 |

## 二、构建 Docker 镜像

```bash
# 1. 准备构建上下文
mkdir -p build/models/bge-small-zh-v1.5 build/data build/wheels
cp -r 研发/* build/

# 2. 下载 PyTorch Linux wheel
pip download torch --only-binary=:all: \
  --platform manylinux_2_28_aarch64 \
  --python-version 311 --implementation cp --abi cp311 \
  --extra-index-url https://download.pytorch.org/whl/cpu \
  --dest build/wheels/

# 3. 下载其他依赖
pip download streamlit pymupdf sentence-transformers faiss-cpu whoosh jieba \
  numpy requests openai -i https://mirrors.aliyun.com/pypi/simple/ \
  --dest build/wheels/ --no-deps

# 4. 复制模型和数据
cp -r ~/.cache/huggingface/.../bge-small-zh-v1.5/* build/models/bge-small-zh-v1.5/
cp data/vector_index.faiss data/chunk_metadata.json build/data/

# 5. 构建
docker build -t finance-qa:latest build/
```

## 三、启动容器

```bash
# 启动
docker run -d --name finance-qa -p 8520:8501 \
  -e DASHSCOPE_API_KEY="sk-your-key" \
  finance-qa:latest

# 查看状态
docker ps | grep finance-qa

# 查看日志
docker logs finance-qa -f
```

## 四、验证部署

```bash
# 验证服务可达
curl -s -o /dev/null -w "HTTP %{http_code}" http://localhost:8520
# 输出: HTTP 200 ✅

# 验证容器内 RAG 问答
docker exec finance-qa python -c "
from qa_engine import RAGSystem
system = RAGSystem(llm_config={})
print(system.ask('注册资本是多少？')['rag_answer'])
"
```

## 六、数据持久化

```bash
# 创建数据卷
docker volume create finance-qa-data

# 使用数据卷启动
docker run -d --name finance-qa -p 8520:8501 \
  -v finance-qa-data:/app/data \
  finance-qa:latest
```
