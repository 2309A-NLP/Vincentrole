# 部署测试

## 测试问题清单（基于招股说明书）

1. 武汉兴图新科电子股份有限公司注册资本是多少？
2. 公司法定代表人是谁？
3. 报告期内公司来自军用领域的收入分别是多少？
4. 军用领域收入占主营业务收入的比重是多少？
5. 电子信息行业的上游涉及哪些企业？
6. 下游主要包括哪些行业？
7. 公司在哪个领域已经成为重要供应商？
8. 公司参与制定了哪个技术标准？
9. 公司参与的哪个工程荣获了国家科技进步一等奖？
10. 公司计划使用本次发行募集资金的多少用于补充流动资金？

## 测试方法

```bash
# 直接在容器内执行 RAG 问答
docker exec finance-qa python test_deploy.py
```

## 部署后测试结果

将 `http://localhost:8520` 在浏览器中打开，逐条提问以上问题，验证：
- [ ] 页面正常加载
- [ ] 问题提交后有响应（非空白）
- [ ] 回答内容与招股说明书一致
- [ ] 响应时间 < 3秒
