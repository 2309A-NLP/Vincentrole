## 3. 部署步骤

### 3.1 安装依赖

```bash
cd 工单5/研发
pip install -r requirements.txt
```

**依赖清单：**

| 包名 | 用途 |
|------|------|
| streamlit | Web 前端 |
| pymupdf | PDF 解析 + 图像提取 |
| faiss-cpu | 向量索引 |
| sentence-transformers | BGE Embedding |
| rank-bm25 | BM25 检索 |
| pymilvus（可选） | Milvus 客户端 |

### 3.2 配置 API

```python
LLM_CONFIG = {
    "提供商": "custom",
    "模型": "qwen-plus",
    "API地址": "https://dashscope.aliyuncs.com/compatible-mode/v1",
    "API密钥": "sk-your-key",
```


**Milvus**：需先启动 Docker 容器：
```bash
docker run -d --name milvus-standalone \
  -p 19530:19530 -p 9091:9091 \

```

## 5. 常见问题

### Q1: 向量后端怎么切换？

```python
VECTOR_STORE_BACKEND = "faiss"   # 本地文件
VECTOR_STORE_BACKEND = "milvus"  # Docker 服务
```
切换后删除旧索引，重启系统重建。

### Q2: 多轮对话不生效？

检查 `CONVERSATION_CONFIG["启用多轮对话"] = True`。前端需要在同一会话中连续提问才有历史。

### Q3: 关键词兜底是什么？

当向量检索返回 0 条时，自动用 BM25 关键词检索追加最多 5 条结果作为保底。

### Q4: 如何重建索引？

```bash
rm -f data/vector_index.faiss data/chunk_metadata.json data/chunk_metadata_bm25.pkl
```
重启系统自动重建。
