## 3. 部署步骤

### 3.1 安装依赖

```bash
# RAG 系统依赖
cd 工单11/研发/rag
pip install -r requirements.txt

# 微调依赖（如需训练）
cd ../embed_finetune
pip install -r requirements.txt
```

### 3.2 配置 API

```python
# rag/config.py
LLM_CONFIG = {
    "提供商": "custom",
    "模型": "qwen-plus",
    "API地址": "https://dashscope.aliyuncs.com/compatible-mode/v1",
    "API密钥": "sk-your-key",
}
```

### 3.3 选择 Embedding 模型

```python
# rag/config.py — 切换 Embedding 模型
EMBEDDING_CONFIG = {
    "模型名称": "finetuned-bge-base-zh-v1.5",  # 使用微调模型
    "模型路径": "../embed_finetune/finetuned-bge-base-zh-v1.5",
    "设备": "cpu",
    "维度": 768,
}
```

如需使用原版 bge-base，切换为：
```python
EMBEDDING_CONFIG = {
    "模型名称": "bge-base-zh-v1.5",
    "设备": "cpu",
    "维度": 768,
}
```

### 3.4 启动 RAG 系统

```bash
cd 工单11/研发/rag
streamlit run app/ui.py
```

## 4. 微调操作指南

## 5. 常见问题

### Q1: 微调需要多长时间？

在 M1 Mac CPU 上约 5 分钟（3 epoch, 1320 样本）。

### Q2: 微调需要 GPU 吗？

不需要，CPU 可运行。有 GPU 会更快。

### Q3: 如何切换到微调模型？

修改 `rag/config.py` 中 `EMBEDDING_CONFIG["模型名称"]` 为 `"finetuned-bge-base-zh-v1.5"`。

### Q4: 微调模型文件太大无法上传？

微调模型约 400MB，建议上传到 HuggingFace Hub 或网盘。
