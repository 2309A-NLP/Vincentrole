# 工单13 — RAG 性能瓶颈识别与优化

## 项目结构

- **研发/** — 优化后的RAG问答系统（TF-IDF重排 + qwen-turbo + FAISS）
- **测试/** — 基线测试 + 优化方案对比脚本及数据
- **优化/** — 最优配置 + 优化前后对比报告
- **设计/** — 架构对比图

## 优化结果

| 指标 | 优化前 | 优化后 |
|------|:------:|:------:|
| 总耗时 | 4,558ms | **595ms** |
| 重排器 | CrossEncoder (~2s) | TF-IDF (~10ms) |
| LLM | qwen-plus (~2.1s) | qwen-turbo (~0.5s) |
| 3秒达标 | ❌ 超52% | ✅ 快5倍 |
