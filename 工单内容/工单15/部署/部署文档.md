# 工单15 — 部署文档

## 系统概述

基于跨模态检索优化的技术图纸问答系统，支持文本+图片混合查询。

## 环境依赖

```bash
pip install openai pymupdf
```

## 项目结构

```
工单15/
├── 研发/
│   ├── config.py                # 配置（问题列表、API Key）
│   ├── cross_modal_retriever.py # 核心：跨模态检索优化
│   ├── test.py                  # 6题自动化测试脚本
│   └── patent_images/           # 专利PDF提取的图片
├── 测试/
│   └── 测试记录.md
├── 优化/
│   └── 优化报告.md
├── 设计/
│   └── 架构图.html
└── 部署/
    └── 部署文档.md
```

## 使用方式

```bash
# 1. 提取专利图片
python -c "
import fitz
doc = fitz.open('path/to/CN100347506C.pdf')
page = doc[10]  # 第11页
pix = page.get_pixmap(matrix=fitz.Matrix(2,2))
pix.save('研发/patent_images/p11.png')
"

# 2. 运行测试
cd 研发
python test.py
```

## API 配置

在 `config.py` 中配置通义千问 API Key：
```python
QWEN_VL = {
    "api_key": "sk-your-api-key-here",
    "model": "qwen-vl-plus",
}
```

## 跨模态检索流程

```
输入问题 → 查询理解(检测图文引用)
         ├─ 纯文本问题 → 文本检索 → qwen-turbo
         └─ 图文问题 → 多路召回(RRF) → qwen-vl-plus → 答案
```
