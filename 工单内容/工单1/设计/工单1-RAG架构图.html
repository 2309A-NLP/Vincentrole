PDF智能问答RAG系统 架构图

工单1 · 基于FAISS + BGE + DeepSeek的检索增强生成问答系统

文档索引流程

  • 1. PDFParser 提取PDF文本（pymupdf/pypdf）
  • 2. TextChunker 智能分块（500字/100重叠）
  • 3. EmbeddingModel 向量化（BGE-base-zh-v1.5）
  • 4. VectorStore 构建FAISS索引并持久化

问答检索流程

  • 1. QueryUnderstanding 意图识别+实体提取+查询重写
  • 2. EmbeddingModel 将问题向量化
  • 3. VectorStore FAISS语义检索 Top-K
  • 4. LLMGenerator 基于上下文生成答案
  • 5. 支持 RAG vs 纯LLM 对比模式

技术栈

  • 前端: Streamlit + Web Speech API
  • Embedding: BAAI/bge-base-zh-v1.5 (768D)
  • 向量库: FAISS IndexFlatIP (余弦相似度)
  • LLM: DeepSeek / OpenAI / Ollama
  • PDF: pymupdf / pypdf 双引擎
