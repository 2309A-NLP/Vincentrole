# 工单14 — 部署与使用文档

## 系统概述

基于 RAGFlow Deepdoc + 通义千问 VLM 的低质量工业 PDF 问答系统。

## 环境依赖

```bash
# Python 3.10+
pip install streamlit pymupdf openai
```

## 运行方式

### 方式一：本地运行

```bash
cd 研发
streamlit run app.py --server.port 8503
```

访问 `http://localhost:8503`

### 方式二：源码结构

```
研发/
├── app.py                      # Streamlit 前端（6题交互测试）
├── deepdoc_技术分析文档.md       # Deepdoc 解析技术说明
```

## API 配置

系统使用通义千问 qwen-vl-plus 进行图片理解，需配置：

```python
# app.py 中的 API 配置
api_key = "sk-your-api-key-here"        # 通义千问 API Key
base_url = "https://dashscope.aliyuncs.com/compatible-mode/v1"
model = "qwen-vl-plus"
```
