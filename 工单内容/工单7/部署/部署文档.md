## 3. 部署步骤

### 3.1 安装依赖

```bash
cd 工单7/研发
pip install -r requirements.txt
pip install whoosh
```

### 3.2 配置 API

```python
LLM_CONFIG = {
    "提供商": "custom",
    "模型": "qwen-plus",
    "API地址": "https://dashscope.aliyuncs.com/compatible-mode/v1",
    "API密钥": "sk-your-key",
}
```

### 3.3 配置数据目录

```python
# config.py
DATA_DIR = "/Users/suwente/Desktop/专高六学习资料/RAG 工单/附件/ccf_competition 2"
SUPPORTED_FORMATS = ['.pdf']     # 支持 PDF，可添加 '.txt'
```

系统启动时自动扫描该目录下所有支持的文件。

### 3.4 启动

```bash
streamlit run app/ui.py
```

## 4. 配置说明

### 4.1 文件解析

```python
FILE_CONFIG = {
    "PDF引擎": "pymupdf",
    "TXT编码": "utf-8",
    "TXT编码回退": ["gbk", "gb2312", "latin-1"],
}
```

### 4.2 检索模式

```python
HYBRID_SEARCH_CONFIG = {
    "默认检索模式": "hybrid",   # vector / fulltext / hybrid
    "向量权重": 0.5,
    "全文权重": 0.5,
    "融合算法": "rrf",          # weighted_avg / rrf / vote
}
```

### 4.3 重排器

```python
RERANKER_CONFIG = {
    "启用重排": True,
    "默认重排器": "crossencoder",  # llm / tfidf / crossencoder
}
```

## 5. 运行评估

```bash
cd 工单7/研发
python test_evaluation.py
```

评估结果输出到终端，包含每道题的正确/错误标记、总正确率、平均响应时间。
